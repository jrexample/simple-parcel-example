# Simple Parcel Example

Simple application with parcel bundler

## Usage
```
npm install -g parcel-bundler
npm install
npm start
```

## Built With
- <a href="https://parceljs.org/">Parcel</a>
- Axios
- Autoprefixer
- jQuery
